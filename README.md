# Minisoft Python API (miniapi)

> Minisoft Python API (miniapi) is a library that allows your Python source code to run smoothly on any platform.With it, your Python source code can run on any platform without modification.
> **Notice** The project is currently in the early development stage, and its features may be unstable.

## Main Uses
- Provide a cross-platform compatibility layer for the Python source code of Minisoft software or other developers.
- Can be used as a plugin API template for Minisoft software or software developed by other developers.

## Main Functions
- File operations
- Process Management
- Internet access
- Software Logging
- Performance Monitoring
- Platform detection
......

## Usage
### Method 1: Imported as a Python library
> First, you need to get miniapi from GitHub or PyPI.
```bash or cmd
# PyPI
pip install miniapi
```
```bash or cmd
# GitHub
git clone https://github.com/Douglas-Woods2023/miniapi.git

pip install -e .
```
> You can import it into your Python source code.
```Python
import miniapi
```
### Method 2: Import as a Minisoft software plugin API
- For details, please refer to the [Minisoft Python API documentation](http://sj.frp.one:40092/miniapi.html).
